# Git

- `git init`
- `git status`
- Working area -> Stage -> Repository
- `git add` `--all`/`*`
- `git commit` `-m` `-am` -> only for files you've already added `--amend -m`
- `git log` `--all --decorate --oneline --graph` (adog)
- `git diff`
- `git branch` `-d` 
- `git push`
- `git pull`
